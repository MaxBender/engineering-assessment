WikiBacon CLI Preview

Deterministic demo runs of the terminal game, styled for README screenshots. The first shows the default category-aware mode. The second shows hard mode, where category shortcuts are ignored and the computed path gets longer.

Normal mode

The default run keeps category links enabled and finds a shorter path through Fruit and Water.

Hard mode

Hard mode ignores category edges, so the computer path has to follow a longer link-only route.
